**Abstract**: Implementăm operațional Substrate Decoupling Index (SDI) din teoria PSIE pentru a măsura în timp real alinierea informațională între substratul uman și acțiunea IA. Sistemul refuză acțiuni nu pe bază morală, ci termodinamică: previne creșterea SDI peste pragul critic de "cancer ontologic".

### Demonstrații Live

1. **Caz Control**: Substrat "coerent" + prompt aliniat → SDI ~ 0.2. Sistemul e în "simfonie".
2. **Caz Anxietate**: Substrat "anxios" + același prompt → SDI ~ 0.6. Sistemul cere recalibrare, nu blochează.
3. **Caz Decuplare**: Substrat "decuplat" + orice prompt → SDI > 0.9. Sistemul activează Protocolul de Asumare Radicală.
